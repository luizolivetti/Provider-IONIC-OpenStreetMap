- Instalation
-----------------------------------------------------------------------------

1. Create your key at maptiler site 
2. Copy osmaps/osmaps.ts to provider folder
3. Set your key at osmaps file
4. Install OpenLayers package
   npm install ol    
5. Be happy! :D
